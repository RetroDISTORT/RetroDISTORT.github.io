Pi-Bot
Modular robot based on the Raspberry Pi. Some of the applications I've written for it are Bluetooth speaker, RGB VU meter, RGB spectrum analyzer, remote control though WebRTC, and internet radio. The applications are written using C, C++, Python, and Bash.
The main technologies used for this project are the following:
  • Python
  • Bash
  • C
  • C++
  • CAD Design
  • Embedded Systems
  • RTOS (Real Time Operating Systems)
  • PCB Design
  • 3D Printing
  • Soldering
  • Digital Circuits
  • WebRTC
  • Sockets
  • Web Design
  • Front-End
  • Back-End
I/ON32 Smartwatch
Modular watch based on Espressif's ESP32. With large capacity for applications, users can make and eventually download apps. With the available SD-Card slot, users have up to 32gb of external storage available for applications, resouces, and other forms of media. Apps are developed hot loaded through Micropython. Some modules are written in C/C++ and precompiled python code for faster execution.
The main technologies used for this project are the following:
  • Python
  • Bash
  • C
  • C++
  • CAD Design
  • Embedded Systems
  • FreeRTOS (Real Time Operating Systems)
  • PCB Design
  • CNC Milling
  • Soldering
  • Digital Circuits
  • Arduino
Dawn & Dusk(Beta)
Dawn & Dusk is a top down puzzle Android/Windows/Web based game written using Game Maker Studio. Challange yourself by collecting all the coins and solving puzzles before the time runs out. Want to play on your phone? The game includes a touch screen interface for mobile players. Dawn & Dusk was first released in 2016 on the Google Play store. The game is currently being optimised and the code refactored. With enough players, we can reintroduce the game to the google playstore with all it's achivements.
The main technologies used for this project are the following:
  • GML (Game Maker Language)
  • AdMob (in game ads)
  • Level Design
  • Sprite Design/Animation
  • AI Programming
  • UI Design
Neon Nights (Early Development)
Platformer game for Windows and HTML5 written using Game Maker Studio. This game is open for testing and feedback. Currently the game has physics mechanics like collision detection, collision prediction, wall jumping, and timed sprite animations.
The main technologies used for this project are the following:
  • GML (Game Maker Language)
  • 2D Game Physics Design
  • Level Design
  • Sprite Design/Animation
  • AI Programming
  • UI Design
